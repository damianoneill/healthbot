Overview

HealthBot is a network health and diagnostic solution that provides consistent and coherent operational intelligence across network deployments.

Integrated with Junos Telemetry Interface (JTI) and standards-based OpenConfig telemetry, HealthBot aggregates large volumes of time series telemetry data and provides a multidimensional view across network and applications, translating real-time analytics into actionable insights.
